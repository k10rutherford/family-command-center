# The Rutherfords — Family Command Center v3.2

Reference-inspired seasonal backgrounds with a clean white calendar layered over the artwork.

Months:
- January: blue winter snow
- February: hearts
- March: clovers
- April: Easter eggs and bunny
- May: pastel stripes
- June: baseball pattern
- July: watercolor flag
- August: school supplies
- September: fall leaves
- October: cute Halloween
- November: pumpkins and bows
- December: Christmas fireplace

Subtle animations are included for snow, hearts, spring drift, July sparkle, and falling leaves.
